XTracks Guide d'Installation
Auteur Okrasa Ghia (okrasa@xtracks.tk)
Traduction Française par Roger

Vous ne construisez pas d'itinéraires, vous n'avez jamais utilisé XTracks avant.
Vous avez juste téléchargé un itinéraire qui indique qu'il est nécessaire
d'installer XTracks pour qu'il fonctionne. C'est pour vous!
(Constructeurs d'itinéraire: téléchargez et lisez la documentation dans les notes
supplémentaires de constructeurs d'itinéraire).
 
1. Faite une copie de sauvegarde du dossier existant 'Train Simulator\GLOBAL'
   quelque part sur votre disque dur.

2. Copiez le contenu du dossier 'XTracks\GLOBAL\SHAPES' dans le dossier de
   'Train Simulator\GLOBAL\SHAPES', remplacez TOUS les dossiers quand une boite
   de dialogue vous le demandera.

3. Copiez le dossier 'XTracks\GLOBAL\TSECTION.DAT' dans le dossier 'Train Simulator\GLOBAL'.
   Vous obtiendrez une question: "voulez-vous remplacer le fichier ...
   par ce fichier possédant une date différente".
   - Si la date du fichier présent sur le disque dur est plus récente que celle
     de mon dossier, vous devrez répondre "Non".  
   - Si mon fichier est plus récent, vous devrez répondre "Oui".   
   - Si vous n'obtenez pas la question vous avez fait quelque chose d'incorrect...

4. C'est fini! Heureux "Railroading" sur votre nouvel itinéraire!

En cas de difficultés, vous pouvez demander assistance sur le Forum de www.train-sim.com
ou www.uktrainsim.com  (d'autres forum peuvent également pouvoir vous aider) ou expédiez
un mail à la personne qui a construit l'itinéraire. Vous pouvez également essayer de
m'expédier un mail (en anglais) ou essayer de lire la documentation XTracks.htm dans
XTracks "route builders extras package". NE PANIQUEZ PAS!
