Projekt Hochflurlinien der Kölner Verkehrs-Betriebe AG ( KVB )      
Version 2.0 ( Build 201x )

#### AddOn für den Microsoft Train Simulator ####

Autor der Strecke: Uwe Knaepel


1. Vorwort:
Die Strecke basiert auf den Stand von 2013 und umfasst in dieser Version folgende Linien:

- Teilstück der Linie 1 ( Aachener Str./Gürtel - Rheinenergie-Stadion -Junkersdorf )
- Linie 5 ( Butzweilerhof - Rathaus - Heumarkt )
- Linie 13 mit einigen Aktualisierungen
- die Linien 16 und 18 sind von ihren Kölner Startpunkten Niehl Sebastianstraße bzw. Thielenbruch
  bis Bonn Hbf fahrbar, auch Fahrten über die Querbahn zwischen Wesseling und Brühl sind möglich.

         
-----------------------------------------------------------------------------------------------------------
2. Systemanforderungen

Für gute Spielbarkeit UND Darstellungsqualität ist mindestens ein Core I3 mit 4GB RAM ( DDR3 ) und eine Mittelklasse-Grafikkarte ( z.B. ATI Radeon HD 6670 )notwendig. Von der Verwendung von OnBoard-Grafikkarten wird abgeraten!

Dieses Addon benötigt zur Funktionalität eine installierte Version des originalen Microsoft Train Simulators (von 2001), inklusive installiertem Update auf Version 1.4.

-----------------------------------------------------------------------------------------------------------
3. Installation Strecke

Die Installation der Strecke sowie des Rollmaterials wird automatisch über das Installationsprogramm vorgenommen.

Da die Speicherverwaltung des originalen Train Simulators nicht sehr gut ist und dieser bei einer Vielzahl an installierten Strecken und Fahrzeugen in die Knie geht, haben wir uns dazu entschlossen, die KVB Hochflurlinien Version 2 in eine so genannte "Parallelinstallation" zu installieren.
Das Installationsprogramm installiert "KVB Hochflurlinien Version 2" in ein eigenes Verzeichnis und kopiert nach Abschluss der Installation der Streckendaten automatisch alle notwendigen Dateien aus der Originalen Train Simulator-Installation. Aus diesem Grund ist es zwingend notwendig, dass der Microsoft Train Simulator auf dem eigenen Computer installiert ist!
Dabei erstellt das Installationsprogramm automatisch eine so genannte "Parallelinstallation" des Train Simulators.

Vor der Installation der Strecke müssen folgende Zusatzprogramme für den Microsoft Train Simulator installiert sein, inklusive Update auf Version 1.4. Ohne diese Programme wird es beim Start des Microsoft Train Simulators zu einem Absturz des Programms kommen.

In diesem Installationsprogramm ist bereits enthalten:
 - X-Tracks Version 3.20 mit Tsection.dat #38
 - NewRoads Version 4
 
#### ACHTUNG - JETZT WIRD ES WICHTIG!!! ####

Nach Abschluss des Installationsprogrammes und dem Kopieren der Daten startet automatisch die Installation des MSTS-Bin-Upgrade (Version 1.8). Hier muss *händisch!!!* der Installationspfad des Programms angepasst werden zum Installationsort der "KVB Hochflurlinien Version 2", z.B.:
c:\programme (x86)\Kleinkoeln\KVB_HF_v2

Ohne die Installation des MSTS-Bin-Upgrades wird das Addon "KVB Hochflurlinien Version 2" nicht ordnungsgemäß starten!

Nach dem Abschluss der Installation wird das Addon über einen entsprechenden Link im Startmenü oder auf dem Desktop gestartet. Wichtig: "KVB Hochflurlinien Version 2" wird NICHT in der Hauptinstallation des Microsoft Train Simulators auftauchen!

WICHTIG: Um eine saubere Installation zu gewährleisten,darf AUF KEINEN FALL ein Virenschutzprogramm aktiv im Hintergrund laufen,
andernfalls kommt es zu eventuell zu Falschdarstellungen oder Fehlermeldungen.
